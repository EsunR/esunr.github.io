Vue父子组件之间的双向通信
2019-12-29|前端Vue|
Word count: 749|Reading time: 2 min

Vue 与 React 为了防止父子组件之间的数据混乱，所以为设计为单项数据流，即父组件仅向子组件传递数据，而子组件无法修改父组件传入的数据，从而影响父组件中的数据。然而在 Vue 中，双向数据流还是可以通过很多方法实现的，比如 v-model 双向绑定就是 Vue 提供的一个数据双向绑定的概念，也是 Vue 的特性之一，除此之外，

1. v-model
