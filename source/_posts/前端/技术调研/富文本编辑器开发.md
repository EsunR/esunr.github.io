重点：

- 文本格式化处理，如加粗、斜体、下划线、删除线、引用、代码块等；
- 丰富节点定义，如图片、表格、链接、视频、音频、文件等；
- undo/redo；
- 协同编辑，wss（websocket）；
- 兼容性，（API， execCommand 弃用）；

数据协议：JSON

```json
{
	"type": "doc",
	"content": [
		{
			"type": "paragraph",
			"text": "Hello, world!",
			"marks": [
				{
					type: "bold",
				}
			]
		}
	]
}
```
